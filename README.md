## Getting Started

### Dependencies

- Javascript: `es2015`

- Node: `10.6.0`

- Dependency Resolver: `NPM v6.6.0+`

---

To get up and running with the **Webpack 4: Split-Config Starter**:

```bash
# Install Dependencies
npm install

# Build
npm run dev
npm run prod
```
